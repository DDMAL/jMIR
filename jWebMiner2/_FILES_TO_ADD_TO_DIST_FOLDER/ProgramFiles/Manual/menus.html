Menu Commands

The jWebMiner software has two menus, each of which can be accessed using either the mouse or via keyboard shortcuts:

1) Search

  • Perform Test Search: This command brings up the Network Search Dialog Box, which can be used to perform searches using a variety of search configurations and to view the details of the individual hits returned as well as overall hit counts. Results for two different web services can be combined side by side. This dialog box is intended primarily for testing and debugging, and is not a central part of jWebMiner's functionality.

2) Information

  • About: Displays version information about jWebMiner.
  • Help: Displays this manual in a window. The left side of the manual can be used to select different topics. Note that the manual may be viewed either via this menu command or with any web browsing application (e.g. Mozilla Firefox). The latter option is preferable, as the jWebMiner online help display is not a fully equipped browser, and may not display external sites linked to by this manual as well as a full-fledged browser.
