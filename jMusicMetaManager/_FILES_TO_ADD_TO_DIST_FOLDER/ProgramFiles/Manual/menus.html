Menu Commands

The jMusicMetaManager software has three menus, each of which can be accessed using either the mouse or keyboard shortcuts:

1) Analysis

  • Restore Default Settings: This command restores the settings in the Options Panel to their startup defaults. This does not affect the three paths at the top of the Options Panel, however.
  • Begin Metadata Analysis: This command initiates processing using the preferences selected in the Options Panel. The user will automatically be notified of any invalid settings when analysis is initiated, and will have the opportunity to correct them at that point before proceeding with analysis and report generation.Selecting this command is the same as pressing the Begin Metadata Analysis Button on the Options Panel.

2) Export

  • Export to Weka ARFF: This command extracts metadata from the iTunes XML and/or MP3 files specified in the Options Panel and exports it to a Weka ARFF file. No processing is applied to the metadata other than possibly the merging of metadata from the two possible sources if both are specified. The user is presented with a dialog box asking where the exported data should be saved to.
  • Export to ACE XML: This command extracts metadata from the iTunes XML and/or MP3 files specified in the Options Panel and exports it to an ACE XML file. No processing is applied to the metadata other than possibly the merging of metadata from the two possible sources if both are specified. The user is presented with a dialog box asking where the exported data should be saved to.

3) Information

  • About: Displays version information about jMusicMetaManager.
  • Help: Displays this manual in a window. The left side of the manual can be used to select different topics.
