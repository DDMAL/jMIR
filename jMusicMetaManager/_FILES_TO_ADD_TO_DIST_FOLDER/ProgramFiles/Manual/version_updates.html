Version Updates

This software is under ongoing development. The changes so far are as follows:

  • 1.2.1: Update to iTunes XML parsing functionality, including the ability to access web parsing functionality through a proxy server.
  • 1.2.1: Updated UtilityClasses support libraries.
  • 1.2: Includes updated support libraries (UtilityClasses and Xerces 2.9.1).
  • 1.01: The effect of the album filter was reversed, so that recordings will be reported as potential duplicates only if they are on different albums when this filter is applied.
  • 1.01: Tutorial, Hints & Suggestions, Problems & Solutions, Codaich and Version Update History sections were added to the manual, as well as the Codaich and OMEN ISMIR papers and various other small additions.
  • 1.01: Verification was implemented to ensure that the user saves report files with a .html extension.
  • 1.01: File streams were closed if an error occurred, thereby preventing some operating systems from marking report folders as busy when errors occurred.
  • 1.01: A bug causing the + sign to be improperly parsed from iTunes files was corrected.
  • 1.01: The clarity of error messages relating to the need to access the internet was improved.
  • 1.01: The About dialog box was updated to reflect the GPL licensing.
  • 1.01: The name of the Information menu was corrected.
  • 1.02: All directory paths in HTML reports use a forward slash instead of the system default.
