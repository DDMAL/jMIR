Using the Software

Running the Software

A file named "jMusicMetaManager.jar" is produced upon installation. The simplest way to start jMusicMetaManager is to double click on this file.

The software may also be accessed via the command line (e.g. the DOS prompt). To access DOS under Windows, for example, go to the Start Menu, select Run and type "cmd". Then use the "cd" command to move to the directory that contains the jMusicMetaManager.jar file. In that directory, type:

java -jar jMusicMetaManager.jar

However, it should be noted that the JRE does not always allocate sufficient memory for jMusicMetaManager to process large music collections. Running jMusicMetaManager using either of the above two methods could therefore result in an out of memory error.

It is therefore preferable to manually allocate a greater amount of memory. 500 to 700 MB should be more than enough for most situations, but one should be careful to leave at least 300MB unallocated under Windows XP. If you have 512 MB of RAM, you should run the jMusicMetaManager software as follows at the command prompt:

java -ms16M -mx200M -jar jMusicMetaManager.jar

If you have 1 GB or more of RAM, you should run jMusicMetaManager as follows:

java -ms16M -mx700M -jar jMusicMetaManager.jar

Using the Software's Interface

The jMusicMetaManager interface is divided into two main panels, which may be accessed via tabs below the menu bar. The first, the Options Panel, allows the user to choose the source(s) of the metadata to analyzed, as well as the settings to be used for metadata error detection and report generation. This Options Panel is itself divided into two subsections, namely the Processing Options section on the left and the Reports Options on the right. The second panel, the Report Panel, can be used to examine generated reports. The menu bar can be used to perform tasks such as restoring default options, exporting metadata to Weka ARFF or ACE XML files, and viewing help information such as the manual. Note that your computer must be connected to the internet and that your firewall must not block Java if you wish to parse Apple iTunes files.

A tutorial is available to give a brief introduction on running the software.

Error Messages and Other Problems

jMusicMetaManager generates informative error messages when any problems occur. The reported problems can most often be resolved by following the instructions in the error message. If a problem is encountered that cannot be resolved in this manner, then the Problems and Solutions section of the manual should be consulted.

The metadata error detection reports generated by jMusicMetaManager and displayed in the Report Panel can be sensitive to the parameters specified in the Options Panel. The Hints and Suggestions section of the manual can be consulted for tips on improving results.
