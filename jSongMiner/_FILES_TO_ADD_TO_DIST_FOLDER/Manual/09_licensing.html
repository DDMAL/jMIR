WEB SERVICES USED BY JSONGMINER

The following web services are used by jSongMiner. Note that users of jSongMiner must acquire their own API keys and enter them into jSongMiner's configuration file in order to access these web services through jSongMiner, as described below:

  • Echo Nest API: The Echo Nest API is a collection of web services that are used by jSongMiner to identify audio and acquire metadata about songs and artists. A free Echo Nest API key must be acquired by each user from http://developer.echonest.com/account/register in order to access the Echo Nest API's services. So, users of jSongMiner must acquire such an API key and enter it into the jSongMiner configurations settings file in order to access Echo Nest web services through jSongMiner. Information on The Echo Nest Developer Network's Terms of Service is packaged with the jSongMiner distribution for reference.
  • Last.FM API: The Last.FM API is a collection of web services that are used by jSongMiner to acquire metadata about songs, artists and albums. The API is described at http://www.last.fm/api. A free Last.FM API key is needed to access the Last.FM web services. In order to acquire one, users must first create a Last.FM account (https://www.last.fm/join), if they do not have one already, and then apply for an API key. The web services may then be accessed in accordance with the Last.FM API Terms of Service, a copy of which is included in the jSongMiner distribution for reference. Users must each acquire their own Last.FM API Key and enter it into the jSongMiner configuration file before they will be able to access Last.FM functionality using jSongMiner.

THIRD-PARTY LIBRARIES USED BY JSONGMINER

The following Java .jar files serve as jSongMiner libraries, and are packaged with jSongMiner:

  • Apache Xerces: The Xerces Java library is used by jSongMiner to parse jSongMiner configuration XML files and ACE XML 1.1 Classifications files. It is distributed under the terms of the Apache License, Version 2.0.
  • Jaudiotagger: The Jaudiotagger library is used by jSongMiner to extract embedded metadata from a variety of file formats. This software is distributed under a GNUL Lesser General Public License, Version 3.0.
  • jEN-api: The jEN-api library is used to send requests and interpret returned results from the Echo Nest API. It is distributed under an OSI BSD license.
  • JSON.simple: The JSON.simple library is used to parse JSON-formatted data. This software is distributed under the Apache License, Version 2.0.
  • last.fm-bindings: The last.fm-bindings library is used to send requests and interpret returned results from the Last.FM API. It is distributed under an OSI BSD license.

THIRD-PARTY EXECUTABLES USED BY JSONGMINER

The following binary executable is used by jSongMiner. For licensing reasons, users must download it themselves, as instructed below, and enter information allowing jSongMiner to access it into the jSongMiner configuration file:

  • ENMFP Code Generator: These libraries and binaries are used to generate fingerprints from audio files that can be sent to the Echo Nest API in order to identify them. A description is available at http://groups.google.com/group/enmfp. They are not included with the jSongMiner distribution, as one must seek permission from The Echo Nest in order to download them, at least as of the time of this writing. They are distributed by The Echo Nest under the Code Generator Library License and Distribution Agreement, available at http://developer.echonest.com/downloads/license, which is itself included for reference with the jSongMiner distribution. After downloading the code generator, users must then enter information into the jSongMiner configuration file so that the software is able to access the code generator.
